# 🚜 Calculadora de Custo Agro

Simulador web interativo para análise de rentabilidade por hectare, focado em cana-de-açúcar e outras culturas do agronegócio.

🔗 **Demo ao vivo:** https://andremathews.github.io/calculadora-custo-agro/

## ✨ Funcionalidades
- 📊 Cálculo automático de custo total, receita, lucro e margem líquida
- 💡 Insights inteligentes: alertas de prejuízo, margem baixa ou operação saudável
- 📱 100% responsivo: funciona em desktop, tablet e mobile
- 💾 Persistência local: seus valores são salvos automaticamente no navegador
- 📈 Gráfico dinâmico com Chart.js para visualização comparativa

## 🛠️ Tecnologias
- HTML5, CSS3 (com CSS Variables), JavaScript (ES6+)
- Chart.js para visualização de dados
- GitHub Pages para hospedagem

## 🎯 Diferencial
Desenvolvido por Andre Mathews, estudante de Engenharia Agronômica com foco em tecnologia para o agronegócio. Este projeto combina conhecimento de domínio (ATR, manejo por talhão, custos de produção) com habilidades técnicas (front-end, análise de dados).

## 🚀 Como usar
1. Ajuste os sliders de custos e receita
2. Veja os resultados atualizarem em tempo real
3. Observe o insight automático com recomendações
4. Seus valores são salvos automaticamente para a próxima visita
